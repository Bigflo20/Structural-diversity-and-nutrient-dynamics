This is the R code used to analyse the effect of vegetation diversity on soil organic carbon and total nitrogen dynamics. Please, do not hesitate to reach out if you need any clarifications.
The R code allows you to perform the following analysis:
1) Basic diagnostics of variables used to compute the vegetation complexity index (VCI)
2) Pairwise correlations among VCI components and AGB
3) Transforming non-normally distributed variables
4) Constructing main and alternative vegetation complexity indices
5) Pearson correlations among selected predictor variables (Fig. S2)
6) Model selection with VIF filtering and model averaging for functional identity/composition variables
7) piecewise structural modeling (pSEM) for main forward conceptual models
8) Bootstrapping analysis for main pSEM
9) piecewise structural modeling (pSEM) for feedback conceptual models
10) Bootstrapping analysis for main pSEM
11) Bivariate relationships analysis
